PARRAFOS:

Durante los últimos años el uso del teléfono móvil ha ido creciendo de manera exponencial, especialmente tras la eclosión de los dispositivos inteligentes. Estos se han convertido en nuestros compañeros de bolsillo (y de vida) que nos acompañan allá donde vayamos. Sin embargo, un uso excesivo y no racional puede ocasionar problemas de dependencia, adicción y miedo.

La nomofobia (non-mobile-phone-phobia) puede entenderse como un miedo o ansiedad extrema de carácter irracional que se origina cuando la persona permanece durante un período de tiempo sin poder usar su teléfono móvil. Bien por no tenerlo al alcance o bien por haber agotado la batería, plan de datos o imposibilidad de conectarse a una red wifi.

CODIGO:

Una clase nodo con su metodos:

                        
                            #pragma once
                            #include <stream>

                            using namespace std;
                            template<class T>
                            class Nodo
                            {
                            private: 
                                T elemento;
                                Nodo<T>* anterior;
                                Nodo<T>* siguiente;
                                int contador;
                            public: 
                                Nodo();
                                ~Nodo();
                                void setElemento(T elemento);
                                void setSiguiente(Nodo* siguiente);
                                void setAnterior(Nodo* anterior);
                                void setContador();

                                T getElemento();
                                Nodo* getSiguiente();
                                Nodo* getAnterior();
                                int getContador();
                            };

                            template
                            inline Nodo::Nodo()
                            {
                                anterior = nullptr;
                                siguiente = nullptr;
                                contador = 1;


                            }

                            template
                            inline Nodo::~Nodo()
                            {
                            }

                            template
                            inline void Nodo::setElemento(T elemento)
                            {
                                this->elemento = elemento;
                            }

                            template
                            inline void Nodo::setSiguiente(Nodo* siguiente)
                            {
                                this->siguiente = siguiente;
                            }

                            template
                            inline void Nodo::setAnterior(Nodo* anterior)
                            {
                                this->anterior = anterior;
                            }

                            template
                            inline void Nodo::setContador()
                            {
                                this->contador = contador + 1;
                            }

                            template
                            inline T Nodo::getElemento()
                            {
                                return elemento;
                            }

                            template
                            inline Nodo* Nodo::getSiguiente()
                            {
                                return siguiente;
                            }

                            template
                            inline Nodo* Nodo::getAnterior()
                            {
                                return anterior;
                            }

                            template
                            inline int Nodo::getContador()
                            {
                                return contador;
                            }
